Sobre a disciplina de Banco de dados.

Banco de Dados I é um curso introdutório na área de Banco de Dados que apresenta aos alunos os fundamentos de Banco de Dados (modelagem de dados, projeto de banco de dados e implementação de banco de dados). Os alunos também estudarão técnicas para gerenciamento, recuperação e alteração de informações. A ênfase será no modelo relacional, atualmente o modelo mais utilizado na indústria. Serão abordados em detalhes aspectos téoricos tais como normalização, processamento de consultas, cálculo e álgebra relacional. Durante a disciplina, o aluno desenvolverá (em grupo) um trabalho que utilizará os tópicos estudados durante o curso e proporcionará experiência na utilização dos conceitos estudados. Analisaremos também os principais Sistemas de Gerenciamento de Bando de Dados existentes no Mercado.
